Convert visitors into WhatsApp leads with a customizable floating chat button.

== Description ==

WhatsApp Chat Button helps businesses start conversations with visitors through a lightweight, conversion-focused WhatsApp button.

Phase 1 foundations in this branch include:

* Modular plugin bootstrap
* Admin, public, and includes architecture
* Text domain loading
* Activation and uninstall scaffolding
* Settings and analytics table foundations for later MVP phases

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/whatsapp-chat-button` directory.
2. Activate the plugin through the `Plugins` screen in WordPress.
