Turn your website visitors into real conversations with a smart WhatsApp chat button.

== Description ==

**WhatsApp Chat Button** helps you instantly connect with your visitors through WhatsApp and convert them into leads.

Instead of losing potential customers, let them start a conversation with a single click.

🚀 Perfect for:

* Freelancers
* Agencies
* Small businesses
* WooCommerce stores

---

### 🔥 Key Features

* Floating WhatsApp chat button
* Click-to-chat (no API required)
* Smart pre-filled messages
* Dynamic variables:

  * {page_title}
  * {url}
  * {site_name}
* Multi-number routing (based on page)
* Button customization (color, text, position)
* Show delay option
* Lightweight and fast

---

### 💡 Why This Plugin?

Most WhatsApp plugins only add a button.

**WhatsApp Chat Button is built for conversions.**

Guide your visitors, start conversations, and turn traffic into real leads.

---

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/whatsapp-chat-button` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings → WhatsApp Chat Button
4. Add your WhatsApp number and message
5. Save and start receiving messages

---

== Frequently Asked Questions ==

= Do I need WhatsApp Business API? =
No. This plugin uses the official wa.me link.

= Can I customize the message? =
Yes. You can set custom and dynamic messages.

= Does it work on mobile and desktop? =
Yes. It opens WhatsApp app on mobile and WhatsApp Web on desktop.

= Can I use multiple numbers? =
Yes. You can assign different numbers to different pages.
